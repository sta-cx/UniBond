# UniBond iOS Scaffold

This directory contains the Swift source tree for the UniBond iOS client.

Current scaffold:
- `UniBond/` app source tree
- `UniBond/UniBondWidget/` widget extension source tree
- `UniBond/UniBondLiveActivity/` live activity source tree
- `UniBondTests/` test target stub

The Xcode project file is intentionally not fabricated here.
